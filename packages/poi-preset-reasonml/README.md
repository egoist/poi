# poi-preset-reasonml

Use ReasonML in your Poi project.

## Install

```bash
yarn add poi-preset-reasonml --dev
```

## Usage

```js
// poi.config.js
module.exports = {
  presets: [
    require('poi-preset-reasonml')(options)
  ]
}
```

## Options

### loaderOptions

Type: `object`<br>
[options](https://github.com/reasonml-community/bs-loader#options) for bs-loader
